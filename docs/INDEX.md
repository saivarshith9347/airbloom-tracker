## 💡 Common Questions

### How do I...

**Start the development server?**
```bash
npm run dev
```
See: [QUICK_START.md](./QUICK_START.md#2-start-development-server)

**Login to the app?**
Use: `admin` / `admin123`
See: [QUICK_START.md](./QUICK_START.md#login)

**Change the theme?**
Click the Sun/Moon icon in navbar
See: [FEATURES.md](./FEATURES.md#1-dark-mode--light-mode)

**Add a new device?**
Edit `src/lib/thingspeak.ts`
See: [QUICK_START.md](./QUICK_START.md#add-device)

**Export data?**
Go to History page, click "Export CSV"
See: [FEATURES.md](./FEATURES.md#7-history--logs-page)

**Deploy to production?**
See: [DEPLOYMENT.md](./DEPLOYMENT.md)

**Customize colors?**
Edit `src/index.css`
See: [QUICK_START.md](./QUICK_START.md#change-primary-color)

**Change home location?**
Edit `src/pages/Index.tsx`
See: [QUICK_START.md](./QUICK_START.md#change-home-location)
